# PLC Dojo -- Level 1: Fundamentals 

## Overview 
Completion of PLC Dojo Level 1 covering the foundational elements of PLC ladder logic programming using Allen-Bradley RSLogix 500.

## Topics Covered 
- Programming Fundamentals (XIO, XIC, OTE instructions; Timer and Counter Instructions; Comparators; Mathematical Operators)
- Digital and Analog IO
- Analog Process Control
- HOA and PID Control
- Seal-in logic and latch/unlatch coils
- Alarms and Notifications
- HMI (Human Machine Interface)
- Communications
- Basic program structure and rung organization 

## Note on Level 1 
Level 1 is a guided fundamentals course where the instructor walks through each concept step by step. Independent project work begins in Level 2. See the Level 2 repository for hands-on applied logic programs built without guidance. 

## Tools Used 
- PLC Dojo Level 1 Curriculum 
- Allen-Bradley RSLogix 500
