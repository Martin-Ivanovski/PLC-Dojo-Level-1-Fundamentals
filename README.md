# PLC Dojo -- Level 1: Fundamentals 

## Overview 
Completion of PLC Dojo Level 1 covering the foundational elements of PLC ladder logic programming using Allen-Bradley RSLogix 500 / Studio 5000. 

## Certification 
Level 1 Complete -- Certificate of completion below. 

## Topics Covered 
- Normally Open (NO) and Normally Closed (NC) contacts 
- Output coils and energize/de-energize logic 
- Timer On Delay (TON) and Timer Off Delay (TOF) 
- Retentive Timer (RTO) 
- Counter Up (CTU) and Counter Down (CTD) 
- Seal-in logic and latch/unlatch coils 
- Basic program structure and rung organization 

## Note on Level 1 
Level 1 is a guided fundamentals course where the instructor walks through each concept step by step. Independent project work begins in Level 2. See the Level 2 repository for hands-on applied logic programs built without guidance. 

## Tools Used 
- PLC Dojo Level 1 Curriculum 
- Allen-Bradley RSLogix 500
